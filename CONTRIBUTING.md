# 🤝 Contributing to Galaxy Defender

Thank you for considering contributing to this project! 🚀
Your help is welcome to improve and enhance the **Galaxy Defender** game.

---

# 📌 How to Contribute

## 🐛 1. Report Bugs

If you find any bug:

* Open an issue 📄
* Describe the problem clearly
* Add screenshots (if possible) 🖼️

---

## ✨ 2. Suggest Features

You can suggest new ideas like:

* 🎯 New enemy types
* 🔫 Power-ups
* 🎮 Better controls
* 🎵 Sound effects

---

## 💻 3. Code Contributions

### Steps:

1. 🍴 Fork the repository
2. 📥 Clone your fork:

```bash
git clone https://github.com/your-username/galaxy-defender.git
```

3. 🌿 Create a new branch:

```bash
git checkout -b feature-name
```

4. ✍️ Make your changes in code

5. 💾 Commit changes:

```bash
git commit -m "Added new feature"
```

6. 🚀 Push to GitHub:

```bash
git push origin feature-name
```

7. 🔁 Create a Pull Request

---

# 🧠 Code Guidelines

* ✔️ Write clean and readable C++ code
* ✔️ Use proper indentation
* ✔️ Comment your logic clearly
* ✔️ Do not break existing game logic

---

# 🎮 Project Structure

* `Display()` → handles rendering 🖥️
* `Timer()` → game logic engine ⚡
* `PrintableKeys()` → keyboard input ⌨️
* `NonPrintableKeys()` → arrow controls 🚀

---

# 🧪 Testing

Before submitting:

* Run the game on Ubuntu 🐧
* Check for crashes 💥
* Ensure controls work properly 🎮
